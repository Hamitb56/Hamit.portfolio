Autonomous Indoor Navigating Robot

PLUG IN

Lidar Mapping & ROS2

This project focuses on the design and implementation of a holonomic autonomous mobile robot optimized for indoor environments using the ROS 2 (Jazzy) framework. The system architecture is built on a hybrid control model where high-level decision-making and sensor processing are handled by a Raspberry Pi 5, while low-level actuator synchronization and real-time tasks are managed by an Arduino Mega 2560.

A significant engineering challenge addressed during development was the compensation for hardware limitations, specifically a faulty wheel encoder and mechanical drift. To overcome this without increasing costs, I implemented a custom virtual encoder algorithm that estimates motion using motor commands and physical constraints, achieving a deviation rate of less than 2%.

The navigation pipeline utilizes a custom-designed dynamic path planning and obstacle avoidance strategy. This system analyzes environmental data across four distinct zones—front, rear, left, and right—to make instantaneous trajectory adjustments. By employing a rule-based reaction matrix, the robot can detect and bypass both static and dynamic obstacles.
